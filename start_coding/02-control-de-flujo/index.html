Ejercicio 2

Numeros pares e impares. Hacer un programa que itere por los numeros del 0 al 10 y nos diga si es par o impar.

Ejemplo:

0 es par

1 es impar

2 es par

...

Está comentado en el archivo script.js, para que al renderizar al web se vea el chalenge 2. Mi forma de hacerlo :

for (let index = 0; index <= 10; index++) { if (index % 2 === 0) { console.log(`${index} es número par`); } else { console.log(`${index} es número impar`); } }

Como el bloquea ejecutar es de una sola declaración, puedo omitir las { }, y me queda así :

for (let index = 0; index <= 10; index++) { if (index % 2 === 0) console.log(`${index} es número par`) } else console.log(`${index} es número impar`); }

Ejercicio 3

FizzBuzz. Generar un programa que itere del 1 al 15. Por cada iteración mostrar or consola lo siguiente:

1- Si el numero es multiplo de 3, mostrar la palabra"Fizz"

2- Si el numero es multiplo de 5, mostrar la palabra "Buzz"

3- Si el numero es multiplo de 3 y 5, mostrar la palabra "FizzBuzz"

4- En los demás casos, mostrar el número

Está comentado en el archivo script.js, para que al renderizar al web se vea el chalenge 2. La forma en que lo hice:

for (let numeroIterado = 1; numeroIterado < 16; numeroIterado++) { if (numeroIterado % 3 === 0 && numeroIterado % 5 === 0) { console.log("FizzBuzz"); } else if (numeroIterado % 3 === 0 ) { console.log("Fizz") } else if (numeroIterado % 5 === 0 ) { console.log("Buzz"); } else { console.log(numeroIterado); } }

Coding Challenge 2

Registro

Vamos a construir un programa que permita al usuario registrarse en en nuestra aplicación. Primero mostrar un mensaje que le de la bienvenida al usuario y le pregunte si desea registrarse. navegador que le permia hacer este tipo de consultas al usuario, devolviendo caso de que el usuario acepte registrarse, le vamos a pedir que ingrese su nombre de usuario que debe validar que tenga por lo menos 10 caracteres y no va a ser case sensitive, por lo que se va a guardar siempre en minuscula , no como lo ingrese el usuario. Agregar la validación necesaria para que en se muestre un mensaje de error.

Si el nombre de usuario es ingresado correctamente, le vamos a pedir al usuario la contraseña de por lo menos de 6 caracteres. La contraseña tiene que ser case sensitive, por lo que se guarda tal cual la ingresa el usuario.

En caso de que la contraseña no cumpla con el requisito anterior, se debe mostrar mensaje de exito informando al usuario que se completó el registro.

La forma de resolverlo:

const wishesToRegister = confirm("Bienvenido al sitio as click en OK para registrarte"); if (wishesToRegister) { let registerUserName = prompt("Ingresa tu nombre de usuario: "); if(registerUserName.length >= 3) { registerUserName = registerUserName.toLocaleLowerCase(); const registerPassword = prompt("Ingresa tu password : "); if(registerPassword.length>= 6 ) { alert("El registro se completó exitosamente"); }else { alert("El password debe tener por lo menos 6 caracteres."); } } else { alert("El nombre de usuario debe tener por lo menos 3 caracteres."); } } else { //si hace click en cancelar para no registrarse alert("Ok te esperamos cuando gustes"); }

Login

Partiendo de la base del punto anterior, en caso de que se complete el registro se le muestra un mensaje al usuario preguntandole si desea iniciar sesion. En caso de que el usuario haga click en aceptar le pedimos que ingrese el nombre de usuario y la contraseña que uso para registrarse. Si alguno de los datos ingresados no coinciden con los informados al momento del registro mostrar un mensaje de error. Caso contrario, le mostramos un mensaje que diga que el login fue exitoso para poder hacer la comparación contra los valores almacenados.

Recordar normalizar los datos para poder hacer las comparaciones

La forma de que lo resolvi:

let isRegistered, registerUserName, registerPassword; const wishesToRegister = confirm("Bienvenido al sitio as click en OK para registrarte"); if (wishesToRegister) { registerUserName = prompt("Ingresa tu nombre de usuario: "); if(registerUserName.length >= 3) { registerUserName = registerUserName.toLocaleLowerCase(); registerPassword = prompt("Ingresa tu password : "); if(registerPassword.length>= 6 ) { alert("El registro se completó exitosamente"); }else { alert("El password debe tener por lo menos 6 caracteres."); } } else { alert("El nombre de usuario debe tener por lo menos 3 caracteres."); } } else { //si hace click en cancelar para no registrarse alert("Ok te esperamos cuando gustes"); } if(isRegistered){ const loginUserName = prompt("Ingresa tu usuario : ").toLowerCase(); const loginPassword = prompt("Ingresa tu password : "); if(loginUserName === registerUserName && loginPassword === registerPassword ){ alert("Login exitoso !"); } else{ alert("Alguno de los datos ingresados es incorrecto."); } } else { alert("Vuelve cuando gustes"); } }

Extra

Tomando los ejercicios anteriores, los vamos a modificar para que en el caso de tener algún mensaje de error, el programa vuelva a pedir al usuario la información solicitada en ese punto, el usuario en cualquier momento puede terminar el programa tocando el boton de cancelar al momento en que se le pide ingresar sesion o cuando le pedimos los datos.

La forma de resolverlo:

let isRegistered, registerUserName, registerPassword; const wishesToRegister = confirm("Bienvenido al sitio as click en OK para registrarte"); do { if (wishesToRegister) { if(!registerUserName){ registerUserName = prompt("Ingresa tu nombre de usuario: "); } if(registerUserName.length >= 3) { registerUserName = registerUserName.toLocaleLowerCase(); registerPassword = prompt("Ingresa tu password : "); if(registerPassword.length>= 6 ) { alert("El registro se completó exitosamente"); isRegistered = true; }else { alert("El password debe tener por lo menos 6 caracteres."); registerPassword = null; } } else { alert("El nombre de usuario debe tener por lo menos 3 caracteres."); } } else { alert("Ok te esperamos cuando gustes"); } } while (wishesToRegister && ( !registerPassword || !registerUserName) ); // La parte del login let loginPassword, loginUserName; if(isRegistered){ let wishesToLogin= confirm("Bienvenido al sitio as click en OK para ingresar - INICIAR SESION -"); do { if(wishesToLogin){ loginUserName = prompt("Ingresa tu usuario : "); if(loginUserName !== null ) { loginUserName = loginUserName.toLowerCase(); loginPassword = prompt("Ingresa tu password : "); if(loginPassword === null) { wishesToLogin = false; } else if ( loginUserName === registerUserName && loginPassword === registerPassword ) { alert("Login exitoso !"); } else { alert("Alguno de los datos ingresados es incorrecto."); loginPassword = null; loginUserName = null; } } else { wishesToLogin = false; } } else { alert("Vuelve cuando gustes"); } } while (wishesToLogin && ( !loginUserName || !loginPassword) ); }
