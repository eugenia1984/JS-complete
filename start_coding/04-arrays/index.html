Arrays

Ejercicio :

Dada la siguiente lista de invitador en ese orden:

Lucas,

Matias,

Jose,

Pedro,

Martina,

Marcelo,

Esteban,

Marcela,

Martin

  1. Informar por consola el número de invitados
  2. Imprimir por consola el nombre del último invitado
  3. A último momento se invitó a Pedro a la fiesta, por lo que hay que agregarlo al final de la lista
  4. Media hora antes de empezar Marcela confirmó que no podía asistir. Generar una nueva lista que contenga los invitados que confirmaron su presencia.
  5. Por cada invitado, asignarle un número de orden confirme su posición en la lista. Mostrar este número de orden en un nuevo array de la siguiente manera: "Juan. orden: 1".
  6. Suponiendo que los invitador consumieron la siguiente cantidad de platos:
    • Lucas: 3 platos
    • Esteban: 1 plato
    • Jose: 1 plato
    • Los restantes: 1 plato
    Calcular el número total de platos consumidos al final de la fiesta.
  7. Acomodar los invitados por orden alfabético.

¿ Cómo lo resolví ?

Primero: creo mi array de invitados

const invitados = [ "Lucas", "Matias", "Jose", "Pedro", "Martina", "Marcelo", "Esteban", "Marcela", "Martin", ];

Segundo: informo el numero de invitados calculando .length de mi array

console.log(`La cantidad de invitados es de: ${invitados.length}`);

Para imprimir el nombre del último invitdo utilizo .length-1 para acceder al último índice del array (que es un número menor al length del array).

console.log(`El último invitado es: ${invitados[invitados.length-1]}`);

Para agregar a Pedro al final de la lista, utilizo le método .push()

invitados.push("Pedro");

Hago un console.log para ver que se agregó pedro la final

console.log(invitados);

Marcela no puede asistir, nueva lista que confirma los invitados que existen. Tengo que sacar a marcela, utilizo un filter

const invitadosConfirmados = invitados.filter( (name) => name !== "Marcela");

Lo imprimo por consola:

console.log(invitadosConfirmados);

Por cada invitado debo generar un número de orden, utilizo el método .map()

const invitadosConNumero = invitadosConfirmados.map( (firstName, index) => ` ${firstName}. Orden: ${index+1}`);

Lo muestro con console.log()

console.log(invitadosConNumero);

Para calcular el total de platos, tengo que ir sumando la cantidad que comió cada uno, usando un reduce.

const totalPlatos = invitadosConfirmados.reduce( (acumulador, valorActual) => { if( valorActual === "Lucas" ) { return acumulador + 3; } else if( valorActual === "Esteban" || valorActual === "Jose" ) { return acumulador +1; } else { return acumulador + 4; } }, 0);

Mando por console.log el total de los platos

console.log(`El total de los platos es de : ${totalPlatos}`);

Para acomodar a los invitados por orden alfabético utilizo el método .sort()

const listaOrdenada = invitadosConfirmados.sort();

muestro la lista con console.log

console.log(listaOrdenada);
